Have you already started work on the project?

Help with when a project starts

You have started if you have committed to major expenditure, like buying equipment or paying a deposit.

You can:

  • start gathering quotes from suppliers and apply for planning permission
  • start construction to retain planning, like putting in foundations and hardcore
  • complete works that are not included in the estimated project costs

You cannot ask for grant funding for work already done.
